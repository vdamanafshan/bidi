bidi package

Support for bidirectional typesetting in plain TeX and LaTeX

What is bidi package?

bidi package is a convenient interface for typesetting bidirectional texts with plain TeX and LaTeX. The package includes adaptations for use with many other commonly-used packages.

For full information, see the documentation.

How can you download the bidi package?

If you have installed a recent TeX distribution (either TeXLive or MiKTeX), then bidi package is a part of your TeX distribution and there is no need to download bidi package separately.

Release versions of bidi package are available from CTAN.

Development version of the bidi package can be downloaded by clicking on the folder icons at the top of this page.

Bug Reports

If you believe that you have found a bug in bidi package, then you may submit your bug reports in the issue tracker.

What should you do before submitting your bug reports?

  • Make sure that it is exactly bidi package's issue;
  • Make a minimal (La)TeX document (using minimum text and minimum number of packages) showing the problem;
  • Do not include your private packages or package which are not included in TeXLive;
  • If you have got to use any font in your bug reports, then use free fonts;
  • Do not include images in your bug reports; instead use demo option of graphicx package to replace all your images by a black rule.

What should you do when submitting your bug reports?

  • Use an appropriate label for your bug reports;
  • Briefly state how should one reproduce the problem you are describing;
  • Describe the problem clearly;
  • State what TeX distribution (MiKTeX or TeXLive) you use? what is the version of your TeX distribution? what is the version of your XeTeX binary? what is the version of bidi package you are using?
